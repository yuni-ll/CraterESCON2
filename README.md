### CraterESCON2 (for drive motors)
#### recommendation: run it in a venv

#### create
```bash
python -m venv .venv
```

#### activate 
- windows:
  ```powershell
  source .venv/bin/activate
  ```
- macOS/linux
  ```bash
  .venv\Scripts\activate
  ```


#### install in editable mode
```bash
pip install -e .
```
